# TSetSpawn
The most advanced SetSpawn plugin

# Features
- SetSpawn
- Spawn
- Spawn on enter
- Welcome messages
- Join/Quit messages
- Cooldown
- Wait-time
- Particles
- Sound on spawn
- Void Teleport (configurable worlds)
- Cancel teleport on move
- Death to spawn
- Keep inventory
- PlaceholderAPI Expansion
- Title on spawn
- First join spawn
- Send a specific player to spawn
- Sends all players to spawn
- Delete spawn & ftspawn with command

# Commands
- /setspawn
- /spawn
- /tsetspawn reload
- /tsetspawn
- /tsetspawn version
- /tss
- /tss version
- /tss reload
- /ftsetspawn
- spawn (player)
- spawn all
- removespawn (spawn/ftspawn)

# Permissions
- tsetspawn.reload
- tsetspawn.version
- tsetspawn.setspawn
- tsetspawn.ftsetspawn
- tsetspawn.admin (/spawn (player) and /spawn all)
- tsetspawn.remove

# Placeholders
- %tsetspawn_spawn%
