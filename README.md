# TSetSpawn
Simple SetSpawn plugin

# Features
- SetSpawn
- Spawn
- Spawn on enter
- Welcome messages
- Join/Quit messages
- Cooldown
- Wait-time
- Particles
- Sound on spawn
- Void Teleport
- Cancel teleport on move
- Death to spawn
- Keep inventory

# Commands
- /setspawn
- /spawn
- /tsetspawn reload
- /tsetspawn
- /tsetspawn version

# Permissions
- tsetspawn.reload
- tsetspawn.version
- tsetspawn.setspawn

# Dependencies
- None
