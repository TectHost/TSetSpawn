# TSetSpawn
The most advanced SetSpawn plugin

# Features
- SetSpawn
- Spawn
- Spawn on enter
- Welcome messages
- Join/Quit messages
- Cooldown
- Wait-time
- Particles
- Sound on spawn
- Void Teleport
- Cancel teleport on move
- Death to spawn
- Keep inventory
- PlaceholderAPI Expansion
- Title on spawn
- First join spawn

# Commands
- /setspawn
- /spawn
- /tsetspawn reload
- /tsetspawn
- /tsetspawn version
- /tss
- /tss version
- /tss reload
- /ftsetspawn

# Permissions
- tsetspawn.reload
- tsetspawn.version
- tsetspawn.setspawn
- tsetspawn.ftsetspawn
