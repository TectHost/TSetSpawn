# TSetSpawn
Simple SetSpawn plugin

# Features
- SetSpawn
- Spawn
- Spawn on enter
- Welcome messages
- Join/Quit messages
- Cooldown
- Wait-time

# Commands
- /setspawn
- /spawn
- /tsetspawn reload
- /tsetspawn
- /tsetspawn version

# Permissions
- tsetspawn.reload
- tsetspawn.version
- tsetspawn.setspawn

# Dependencies
- None
